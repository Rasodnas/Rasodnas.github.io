Collaborative online project made with Processing Java Script libraries, for the Creative Coding course at Politecnico di Milano, along with Federico Cordelli, Andrea Gaglione and Francesco Invidia.

Yourpublictoilet.com

gives you access to large HTML canvases called

toilets

, where you can anonymously post text-based messages. Each toilet represents a specific geographical region, where users can leave their mark only if geolocated inside it in real life. The idea behind the project was to create a virtual space that would encourage people from a specific state, region or city to communicate publicly but out of the standards of online communications. The code is in a beta state but currently offline.
Main features

– Each toilet represent a geographical region (in the original release: all 20 italian regions)
– You can view and post on toilets, only if you are geolocated inside the region
– The text you have posted fades over time
– The toilets are divided in numbered tiles for orientation
– You can quickly go to a specific tile
